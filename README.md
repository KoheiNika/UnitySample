# UnitySample
ファイルが多すぎたので圧縮してアップロードします


動作環境は64bitのWindowsを想定しています。食べ物を運ぶアルバイトをしている女の子がビルの屋上から落下しないようにジャンプし続けるゲームです。操作方法は、クリックしてジャンプ、エディットモードではホイールクリック＆ドラッグでスクロール、以上です。ゲーム内でチュートリアル的なものや説明は入れましたが、どうすれば読みやすいようになるかわかりませんでした。ゲーム性が単純だったこともあり、あったほうがいいと思った機能は思いつく限り実装しました。

Unityは2年ほど前に公式チュートリアルのRoll a Ball、Survival Shooterをやったきりで、操作を思い出しつつ、公式リファレンスや質問サイトを参照しながら作りました。かかった期間は、素材の作成含めて大体２０日です。今回学んだことは、やはりゲームでも最初の設計が大切だということでした。こんなゲームが作りたい、という理想はあり、今回はなんとかその通りのゲームを作れましたが、小規模かつ自分が描いたコードだから何とかなった場面が多かったです。リーダブルコード等を読んでネーミング規則を意識していたものの、開発の後半になるほど「もっとましな名前つけるべきだった」という事態が多発しました。また、実際のゲーム開発にできるだけ近いことをしようと思い、Assetなどをできるだけ使わず、素材を自分でUnityで使えるようにするところから行いました。この点に関しては、達成できてよかったです。
利用させていただいたフリー素材の効果音はタイトル画面の右下に表記してあります。また、改変Oｋとのことでしたので、Audacityで少し尺を削ったりしました。
